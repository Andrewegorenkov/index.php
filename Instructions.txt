Страница install.php служит для добавления пользователей из файла mates.json.
Файл mates.json должен находиться в корневой папке.
После запуска страницы install.php выведутся все добавленнные данные пользователей.
Файл mates.json я решил не удалять и не переименновывать после всех действий с ним, поэтому повторный запуск 
приведет к дублированию пользователей в БД.
Добавление происходит в таблицу 'user' БД 'test'.

На странице index.php предоставленна форма для добавления новых пользователей и таблица с данными из БД.
Возможен поиск по таблице и фильтрацию по убыванию/возрастанию по любому столбцу.
При нажатии "удалить" пользователь удалится из таблици и БД.
При нажатии "редактировать" все данные пользователя занесутся в форму добавления и кнопка добавить
заменится кнопкой сохранить.
